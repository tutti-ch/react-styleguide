Dark spinner:

```jsx
<div>
    <Spinner size={Spinner.SIZE_SMALL} color={Spinner.COLOR_DARK} />
    <Spinner size={Spinner.SIZE_MEDIUM} color={Spinner.COLOR_DARK} />
    <Spinner size={Spinner.SIZE_LARGE} color={Spinner.COLOR_DARK} />
</div>
```

Light spinner:

```jsx
<div style={{ backgroundColor: '#dbdbdb' }}>
  <Spinner size={3.5} color={Spinner.COLOR_LIGHT} />
</div>
```
