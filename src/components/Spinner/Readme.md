Dark spinner:

```jsx
<div style={{ textAlign: 'center', padding: '1.5%' }}>
    <Spinner size={Spinner.SIZE_SMALL} color={Spinner.COLOR_DARK} />
    <Spinner size={Spinner.SIZE_MEDIUM} color={Spinner.COLOR_DARK} />
    <Spinner size={Spinner.SIZE_LARGE} color={Spinner.COLOR_DARK} />
    <Spinner size={11} color={Spinner.COLOR_DARK} />
</div>
```

Light spinner:

```jsx
<div style={{ backgroundColor: '#dbdbdb', padding: '1.5%', textAlign: 'center' }}>
  <Spinner size={Spinner.SIZE_SMALL} color={Spinner.COLOR_LIGHT} />
  <Spinner size={Spinner.SIZE_MEDIUM} color={Spinner.COLOR_LIGHT} />
  <Spinner size={Spinner.SIZE_LARGE} color={Spinner.COLOR_LIGHT} />
  <Spinner size={11} color={Spinner.COLOR_LIGHT} />
</div>
```
