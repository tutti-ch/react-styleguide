# tutti.ch Styleguide

## How to start

```
git clone https://github.com/tutti-ch/react-styleguide.git
cd react-styleguidist/examples/sections
npm install
npm run styleguide-server
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
